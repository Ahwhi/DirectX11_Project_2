## 📘 DirectX11 프로젝트2: 야구

● 본 프로젝트는 **DirectX11 Raster Tek 프레임워크 활용 프로젝트 2**로, 기술적인 구현 경험을 목표로 한 프로젝트입니다.

● Release x64 전용

---

### 🕹️ 조작법
● (방향키) 카메라 이동

● (SHIFT키) 공 던지기

● (L-CTRL키) 점프 애니메이션

● (*NUM1) 타이틀 씬에서 게임 씬으로 이동
